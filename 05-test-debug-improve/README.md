## Analytics Tools
After you ship, you need to know what users are actually doing.

| Tool | Best For | Free Tier |
|------|----------|-----------|
| [PostHog](https://posthog.com) | Open-source, self-hostable product analytics | Free (1M events/mo) |
| [Google Analytics](https://analytics.google.com) | Web traffic, acquisition | Free |
| [Mixpanel](https://mixpanel.com) | Event-based user analytics | Free (20M events/mo) |
| [Microsoft Clarity](https://clarity.microsoft.com) | Session recordings, heatmaps | **Free** |
| [Amplitude](https://amplitude.com) | Product analytics, retention | Free starter |

**Recommendation for Hult Prize teams:** Start with **PostHog** (open source, privacy-friendly) or **Microsoft Clarity** (completely free heatmaps + recordings).

## User Feedback — The Mom Test
📖 *The Mom Test* by Rob Fitzpatrick is the go-to book for running customer interviews that actually tell you the truth.

**Core rules:**
1. Ask about their **life**, not your idea
2. Ask about **specific past examples**, not opinions about the future
3. Talk less, listen **more**
4. Ask about their **problems** — not your solution

- 🔗 [The Mom Test — Buy the book](https://www.momtestbook.com/)
- 🔗 [The Mom Test Summary (PDF)](https://www.slideshare.net/xamarin/the-mom-test-rob-fitzpatrick)
