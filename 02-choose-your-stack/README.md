## 🤖 AI Prototyping Tools

These tools let you go from idea → working app using natural language prompts.

| Tool | Best For | Free Tier | Link |
|------|----------|-----------|------|
| **Lovable** | Non-technical founders, full-stack MVPs | 5 credits/day | [lovable.dev](https://lovable.dev) |
| **v0 by Vercel** | React UI components, Next.js apps | $5 credits/mo | [v0.dev](https://v0.dev) |
| **Bolt.new** | Multi-framework prototyping, fast iteration | 1M tokens/mo | [bolt.new](https://bolt.new) |
| **Replit Agent** | Full dev environment, learning to code | Free tier | [replit.com](https://replit.com) |
| **Manus** | Autonomous AI agent for complex tasks | Waitlist | [manus.im](https://manus.im) |
| **Google AI Studio** | Free Gemini API access, prototyping | **Free** | [aistudio.google.com](https://aistudio.google.com) |

### When to use which?
- **No technical background** → Lovable
- **Want to learn coding while building** → Replit
- **Frontend components fast** → v0
- **Multi-framework flexibility** → Bolt.new
- **Free + powerful** → Google AI Studio

## 💻 Traditional Dev Stack (from slides)

### VS Code + GitHub Copilot
The classic combo for developers who want AI assistance without leaving their IDE.
- 🔗 [VS Code](https://code.visualstudio.com/) — free code editor
- 🔗 [GitHub Copilot](https://github.com/features/copilot) — AI pair programmer ($10/mo, free for students)
- 🔗 [GitHub Student Pack](https://education.github.com/pack) — free Copilot + more for students
